# Task ui

## Description
Improve the UI for the task board.

## Task card
This is what's shown in the columns. It must have:
- Title
- truncated description in a lighter color
- horizontal divider line, not reaching the borders of the card
- below the line, to the left, a person emoji and the name of the assignee (if any). To the right, the date of latest modification

## Task modal
This is what pops up when clicking on a card
- ESC should close it
- Top should be title
- Below should be status with 4 buttons aligned horizontally (Not Started, In Progress, For Review, Done). Clicking the button will update the status.
- Certain statuses require certain fields to be complete. The backend will respond with errors. Show them clearly in the UI.
- Description field. Editable.
- Assignee + session ID. Editable.
- Add comment input
- Comments section
